<p>Meta descriptions don't directly affect search rankings Google has confirmed this. But they do affect click-through rate, which affects how much traffic you get from rankings you already have. A post sitting at position 3 with a compelling meta description can outperform a position 2 result with a weak one. Here's how to write them properly.</p>

<figure><img alt="Writing SEO meta descriptions for Google clicks" src="https://images.unsplash.com/photo-1432888498266-38ffec3eaf0a?w=1200&q=80" style="border-radius: 10px; margin: 20px 0px; max-width: 100%;" /></figure>

<h2>The Technical Rules First</h2>
<p>Keep meta descriptions between 150 and 160 characters. Longer descriptions get cut off in search results. Include your target keyword Google bolds matched keywords in the search snippet, which draws attention. Write in active voice.</p>
<p>One thing worth knowing: Google rewrites meta descriptions about 70% of the time based on what it thinks matches the search query. This doesn't mean you shouldn't write one when Google uses your description, it's because you wrote something better than what it would generate. And the effort still helps in cases where Google does use it.</p>

<h2>What Makes a Description Actually Click-Worthy</h2>
<figure><img alt="SEO click through rate optimization strategy" src="https://images.unsplash.com/photo-1551288049-bebda4e38f71?w=900&q=80" style="border-radius: 10px; margin: 20px 0px; max-width: 100%;" /></figure>
<ul>
  <li><strong>Answer the query directly.</strong> If someone searches "how long does it take to learn Python," the description should give them the answer (or a credible partial answer), not just promise that the post will tell them.</li>
  <li><strong>Be specific.</strong> "Learn the 5 factors that affect how quickly you'll learn Python" is better than "Find out how long it takes to learn Python."</li>
  <li><strong>Create mild urgency or curiosity.</strong> Not clickbait but a description that implies the post contains something the searcher wants to know rather than just confirming the topic.</li>
  <li><strong>Speak directly to the searcher.</strong> "You'll learn..." or "Here's what to do when..." performs better than third-person description.</li>
</ul>

<h2>Using AI to Write Meta Descriptions</h2>
<p>Paste your blog post intro into ChatGPT and ask: "Write 3 meta description options for this post, each under 160 characters. Target keyword: [keyword]. Make each one specific and click-worthy rather than generic." Pick the best one and adjust. This takes two minutes and is worth doing for every post you publish.</p>
